# OneWireTemperatureSensor

Using one wire on Raspberry Pi with DS18B20 temperature Sensor

## Sample run

    term% thermo
    Opening GPIO...

    Reseting...
    Presence detected


    Reseting...
    Presence detected

    Reading Scratch...
    0 -> 10111111
    1 -> 00000001
    2 -> 01001011
    3 -> 01000110
    4 -> 01111111
    5 -> 11111111
    6 -> 00001100
    7 -> 00010000
    8 -> 01100001
    Temp (degree C) = 27.937500
